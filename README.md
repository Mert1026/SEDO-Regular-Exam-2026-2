Попринцип в условието пишеше за .yml файла,
че трябва да тригърне други бранчове, 
но аз го напраих за main тестово само,
защото други няма тук.
Иначе трябва да е така знам, просто за да си го тествам написах да е на main първо и после го оправих:
on:
  push:
    branches:
      - develop
      - 'feature/**'
